Replaces the standard WordPress 'Howdy' welcome with 'Aloha'.

== Installation ==

1. Upload 'howdy-2-aloha' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
